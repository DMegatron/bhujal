# Installation Instructions:
# 1. Make sure you have Node.js (v16 or higher) installed
# 2. Navigate to the client directory: cd client
# 3. Run: npm install
# 4. For development: npm start
# 5. For production build: npm run build

# Runtime Requirements:
# - Node.js v16.x or higher
# - NPM v8.x or higher
# - Modern web browser with ES6+ support

# Development Scripts:
# - npm start: Start development server (http://localhost:3000)
# - npm run build: Create production build
# - npm test: Run test suite
# - npm run eject: Eject from Create React App (irreversible)

# Environment Variables Required:
# - REACT_APP_API_URL (Backend API URL, e.g., http://localhost:5000)
# - REACT_APP_MAPBOX_TOKEN (optional, for enhanced maps)
